# Find_the_number
A game to find a number
A game for learning numbers. First it asks the user to guess a number between 1 and 10 as the computer takes one number randomly. If the user can not find, the computer gives directions. Once the number is found, the user must think and type a number in that range, then the computer will guess that number.Again the user must give directions. So at the end the one who found the number with less tries will be a winner. Easy to learn numbers, So kids among age 5-8 can learn from it..
